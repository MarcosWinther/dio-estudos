Posicionamentos e Exibição CSS display

-> Propriedade Display

* A propriedade CSS display
- A propriedade display vai determinar como as caixas dos elementos devem se comportar no momento de serem exibidas na tela:
-- block
-- inline
-- inline-block
-- none
-- flex
-- grid

* Todo elemento possui um display?
- Todo elemento HTML já vem com um valor de display como padrão.
- Cada elemento HTML já possui, por padrão, um valor da propriedade display já definidas. A tag <div>, por exemplo, possui um display em bloco, enquanto a tag <span> possui o valor inline como padrão.

------------------------------------------------------

-> A propriedade CSS vertical-align
- Essa propriedade permite alinhar verticalmente nossos elementos que estão lado a lado (inline-*)
-- middle
-- baseline
-- bottom
-- top
-- sub
-- super

-----------------------------------------------------

-> Flexbox vs Grid

* Display: flex
- O Flexbox é uma forma de organizar os elementos em uma página dentro de containers e de forma dinâmica, organizando os elementos em uma dimensão: ou horizontalmente (as linhas) ou verticalmente (em colunas).

* Display: grid
- No CSS conseguimos trabalhar o nosso layout utilizando o conceito de Grids. O Grid Layout vai organizar os elementos das nossas páginas em duas dimensões: de forma horizontal e vertical, definindo linhas e colunas para que os conteúdos da página sejam posicionados.
